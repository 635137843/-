# MyBlog
### 项目背景
> 由于疫情期间无法外出，在再学习了springboot相关技术，所以打算用所学内容搭建一个属于自己的博客系统，可以自定义分类，标签，发博客等操作，游客可以自由评论或提问。以后可以将自己所学的知识以博客的形式记录下来并分享，也有助于自己复习。
### 采用框架：SpringMVC SpringBoot JPA框架
### 渲染技术: Thymeleaf
